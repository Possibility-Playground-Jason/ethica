Ethica

AI ethics compliance checking as a service, powered by Cloudflare Pages.

API Endpoints

MethodPathDescription
GET/healthHealth check
GET/frameworksList frameworks
POST/checkJSON compliance results
POST/check/reportHTML report card
POST/check/badgeSVG compliance badge
GET/badge/:owner/:repoREADME-embeddable badge

Quick Example

curl -X POST https://ethica.pages.dev/check \
  -H "Content-Type: application/json" \
  -d '{"repo_url": "https://github.com/owner/repo.git"}'
